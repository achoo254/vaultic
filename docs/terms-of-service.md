## 2. Description of Service

Vaultic is an open-source, zero-knowledge password manager that:

- Stores and manages passwords, credentials, and secure notes
- Provides autofill functionality for login forms
- Generates strong passwords
- Offers optional encrypted cloud sync across devices
- Enables encrypted credential sharing via secure links

Vaultic is a password management tool. It is not financial, legal, or security advisory software.
