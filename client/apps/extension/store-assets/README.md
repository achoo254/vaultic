**How to capture:**
1. Build: `pnpm build` from extension directory
2. Load unpacked in Chrome (`chrome://extensions/`)
3. Create demo vault items (Google, GitHub, Netflix, etc.)
4. Screenshot each screen (380x520 popup)
5. Composite onto 1280x800 canvas with clean white/gray background + caption
