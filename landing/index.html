Features

🔗

☁

Sync encrypted vault across devices. Off by default — enable only when you want it. Disable anytime and purge server data.

📡

Offline-First

Full functionality without internet. Your vault is always accessible locally, synced when you choose.

Zero-Knowledge Security

Your master password never leaves your device. The server stores only encrypted blobs it cannot decrypt.

  1. Master password → Argon2id (64 MB, 3 iterations) → master key
  2. HKDF-SHA256 derives per-purpose encryption & auth keys
  3. AES-256-GCM encrypts each vault item (random 12-byte nonce)
  4. Server stores only ciphertext — zero decryption capability
  5. All communication over HTTPS (TLS 1.2+)
